# 💰 FinTracker - Personal Finance Manager

A modern, full-stack **Personal Finance Tracker** built with the **MERN Stack** (MongoDB, Express, React, Node.js). FinTracker helps individuals manage their income and expenses, while providing a powerful administrative dashboard to monitor the platform.

## ✨ Key Features

- **🔐 Authentication & Security:** Secure login/registration using JWT and HTTP-only cookies.
- **👥 Role-Based Access Control (RBAC):**
  - **User Portal:** Track personal income, expenses, view balance, and manage profile settings.
  - **Admin Panel:** Monitor platform-wide transactions, view total users, and manage (update/delete) registered accounts.
- **📊 Interactive Dashboard:** Beautiful, responsive UI with real-time updates and dynamic categorizations.
- **🖼️ Profile Management:** Update personal information, change passwords, and upload profile pictures.
- **🌗 Theme Support:** Built-in Dark and Light mode functionality.
